# a-game

Monorepo for building a **profitable hyper-casual iOS game**. Everything related to
the effort — research, concept validation, prototypes, and the shipped game — lives
here so context is never scattered.

> Working context for AI assistants and collaborators is in [CLAUDE.md](CLAUDE.md).
> Read that first. A scope can carry its own `CLAUDE.md` that Claude Code auto-loads
> when working in that directory.

## Layout

| Path        | What lives here                                            |
| ----------- | --------------------------------------------------------- |
| `DOCS/`     | Decisions, learning paths, research, and pipeline notes.  |
| `.claude/`  | Project-scoped Claude Code config and custom skills.      |

Structure stays intentionally flat for now; new scopes get added only when there's
real material to put in them.

## Working principle

Find an emerging trend with moderate competition, build a **better version**, and
ship. Validation and market fit over complexity. See [CLAUDE.md](CLAUDE.md) for the
full philosophy and current decision log.

## Status

Stage: **trend scouting / validation.** Game concept not yet locked in.
